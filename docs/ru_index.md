# Приведствуем в Unminal Engine!
> 📘 Сейчас вы видите перед собой оффициальную документацию по Unminal Engine

### **Unminal Engine — Freedom to Will, Freedom to Act**
* The engine **doesn't dictate rules**; it hands you the tools to **bring your vision** to life. From a **flexible console system** to **rendering control** — there are no restrictions here, only possibilities.
* Help us keep this spirit alive: **star the repo** and **watch Unminal evolve!**
* A simple, open-source 3D game engine built from scratch in C# using OpenTK.

## История проекта

Изначально Unminal задумывался как хоррор-игра во вселенной [*The Backrooms*](https://en.wikipedia.org/wiki/The_Backrooms)
Но увидев множество подобных проектов, автор решил: **«Зачем делать еще одну игру, если можно создать инструмент для тысяч других?»**

Так родился **Unminal Engine** (ранее EOCS) - инди-движок, написанный из любопытства, переросший в полноценную платформу для творчества
